Basic CREATE TABLE statement

A very basic 

CREATE TABLE
statement which should work in any SQL database

mysql> CREATE TABLE mitabla (
id MEDIUMINT NOT NULL AUTO_INCREMENT,
nombre CHAR(30) NOT NULL,
edad INTEGER(30),
salario INTEGER(30),
PRIMARY KEY (id) );

Note that beginning with MySQL 4.1 

ENGINE=innodb
is the preferred method of defining the storage type

Use 

SHOW CREATE TABLE
(see below) to check that MySQL has created the table as you definied it.

Creating a table with auto_increment

Often you'll want to be able to automatically assign a sequential value to a column:

mysql> CREATE TABLE 'tableT1' (
'idT1' int(11) NOT NULL AUTO_INCREMENT,
'nombreT1' varchar(30),
PRIMARY KEY ('idT1')
);
    
mysql> SELECT id, nombre FROM empleados ORDER BY RAND() LIMIT 3;
+----+----------+
| id | nombre   |
+----+----------+
|  2 | Laura    |
|  7 | Carlos   |
|  4 | Ana      |
+----+----------+
3 rows in set (0.00 sec)
